Script Dromi con PIP

Usiamo un modulo esterno 👇

1️⃣ Codice

from mdl import windawk

windawk.windawk()

2️⃣ Salva il file

Salva il codice in un file chiamato: windawk.dr

3️⃣ Installa il modulo

Apri il terminale nella cartella del file e scrivi: 

dromi.exe --mdl pip install windawk

Premi invio per installare il modulo.

4️⃣ Esegui lo script

Sempre nel terminale, esegui: 

dromi.exe windawk.dr

🎉 Vedrai la tua app con il modulo PIP!
