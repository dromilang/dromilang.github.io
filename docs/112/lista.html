Lista della Spesa

Creiamo una semplice app per aggiungere elementi alla lista 👇

1️⃣ Codice

set_window_title("Lista della Spesa Dromi")
set_window_size(600, 700)

h1("La Mia Lista della Spesa")

input_spesa = inputer()

def aggiungi(elemento):
    p(elemento)
    update()

button("Aggiungi", lambda: aggiungi(input_spesa))

dr.init()

Questo codice crea un campo di input e un bottone. Quando clicchi "Aggiungi", il testo viene mostrato nella lista.

2️⃣ Salva il file

Salva il codice in un file chiamato: lista.dr

3️⃣ Esegui lo script

Apri il terminale nella cartella del file e scrivi: 

dromi.exe lista.dr

🎉 Ora puoi aggiungere elementi alla tua lista della spesa!
