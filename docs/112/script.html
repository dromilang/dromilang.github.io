Primo Script Dromi

Creiamo la tua prima app 👇

🚀 Esempio Completo

set_window_title("Demo")
set_window_size(800, 500)

h1("Benvenuto!")
p("Questa è una app Dromi")

button("Clicca qui")
inputer()

run()

💾 Salva il file

Copia il codice sopra e salvalo in un file chiamato: il_mio_primo_script.dr

▶️ Esegui lo script

Apri il terminale nella cartella dove si trova il file e scrivi: 

dromi.exe il_mio_primo_script.dr

Premi invio e vedrai la tua prima app Dromi! 🎉
