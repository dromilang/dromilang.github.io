Risoluzione dei problemi

Problemi comuni e come risolverli 👇

dromi.exe non funziona

Prova a usare dromi oppure una versione specifica come dromi123.

Se non funziona nulla, reinstalla Dromi seguendo bene le istruzioni.

Non riesco a eseguire gli script

Assicurati di essere nella cartella giusta nel terminale.

Devi eseguire il comando dove si trova il file .dr.

Eseguo lo script ma non succede nulla

Probabilmente non hai salvato il file.

Premi CTRL + S e riprova.

Errore durante l'esecuzione

Controlla il codice: potrebbe esserci un errore di sintassi.

Se non riesci a risolvere, cerca l'errore online o chiedi aiuto.

Non funzionano i moduli

Assicurati di importarli correttamente.

Se hai usato: dromi.exe --mdl pip install nome_modulo

allora devi scrivere: from mdl import nome_modulo
