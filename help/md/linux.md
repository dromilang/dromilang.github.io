# Dromi per Linux

**Dromi per Linux** è la versione per Linux di Dromi.

Puoi usare Dromi per creare programmi, testare idee e imparare come funzionano i linguaggi di programmazione, anche su Linux!

## Come la scarico?

Clicca [qui](https://github.com/DeMENIGECO/Dromi/raw/main/dromi-linux/dromi-linux.tar.gz) per scaricarla

## Perchè non ci sono le librerie preinstallate?

Perchè la build è automatica e non contiene le librerie preinstallate. Puoi leggere come intallarle nel file README.txt in dromi-linux/notes maggiori informazioni. ([Clicca qui](https://github.com/DeMENIGECO/Dromi/blob/main/dromi-linux/notes/README.txt))

## Dove sono le cartelle come mdl, pip_installed o opt?

Semplicemente non sono nel pacchetto. Puoi scaricarle dalla repository manulamente.

## Ci sono altre cose da leggere prima dell'utilizzo?

Sì. Puoi leggerle nel file README.txt in dromi-linux/notes maggiori informazioni. ([Clicca qui](https://github.com/DeMENIGECO/Dromi/blob/main/dromi-linux/notes/README.txt))

# Nota importante!

Scarica tutto il contenuto della cartella notes nella cartella di estrazione di Dromi per eviatre problemi legali e di altro tipo. ([Clicca qui](https://github.com/DeMENIGECO/Dromi/tree/main/dromi-linux/notes))
